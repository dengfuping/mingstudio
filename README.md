# mingstudio
这是基于前端框架Bootstrap编写的一个工作室介绍网站

# 项目说明

- 运行环境：浏览器
- 前端框架：Bootstrap/jQuery
- jQuery插件：单页面导航插件（ `jquery.singlePageNav.min.js` ）、过渡动画插件（ `wow.min.js` 和  `animate.css` ）

# 项目部署

- mingstudio: [http://www.mingstudio.top](http://www.mingstudio.top)
